Question 6

Option (I) casts int 2 to the double 2.0. Upon dividing by 4, the result will be 0.5, so 3.5 is produced
Option (II) casts int (2 / 4) to the double 0.0. The result will thus be 3.0, not 3.5
Option (III) casts int (2 / 4 + 3) to the double 3.0. The result will thus be 3.0, not 3.5
Option (I) is the only option that produces 3.5, so A is our answer.

Question 39

For s1.equals(s2), if s1 > s2, the value returned is positive. In order for the method to return true, the substrings from 0 to 1 must be greater than the substrings from 1 to 2. In other worse, the first character must be greater than the second character for each substring. Thus, the parameter we're looking for must have characters that decrease in lexicographical value. "edcba" is the only parameter that fits this description, so the method call recurMethod("edcba") is the correct answer.
